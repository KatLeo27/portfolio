# Personal Portfolio Website - Kathy Jane Leo

A minimalist personal portfolio website for a Computer Science undergraduate specializing in backend systems, AI/ML, and full-stack development. The site features smooth micro-interactions, responsive grids, interactive architecture flowcharts, and custom canvas-based animations.

Deployed on Vercel.

---

## 🛠️ Tech Stack

- **Core & Logic**: React 18, TypeScript, HTML5
- **Styling & Theme**: Vanilla CSS (Custom Design System Tokens)
- **Animations**: Framer Motion
- **Icons**: Lucide React + Custom Inline SVGs
- **Build Tool**: Vite
- **Asset Processing**: PyMuPDF (for rendering PDF certificates into PNG assets)
